## Implemented Features

### 1) Data Engineering
- Synthetic clinical dataset generation with fields:
  - age
  - gender
  - symptom_*
  - symptom_duration
  - smoking/alcohol flags
  - disease label
- Data cleaning + normalization + encoding prep.
- Feature selection metadata output.
- Final CSV creation at `data/dataset.csv`.

### 2) Machine Learning Pipeline
- Trains and compares:
  - Logistic Regression
  - Random Forest
  - SVM
  - XGBoost (if installed)
- Reports: accuracy, precision, recall, F1-score.
- Automatically selects best model by weighted F1.
- Stores complete model bundle in `backend/model.pkl`.

### 3) Explainability
- Computes feature/symptom importance from best model.
- Returns top contributing symptoms in API response.
- Exposes global model symptom importance in `/model-info`.

### 4) Backend API (FastAPI)
- `GET /health` → API and model health details.
- `GET /metadata` → supported symptoms and diseases.
- `GET /model-info` → model comparison metrics + top global symptoms.
- `POST /predict` → top-3 diseases with confidence + important symptoms.

### 5) Frontend Dashboard
- Professional two-column web UI.
- Searchable symptom selector.
- Input validation and request status handling.
- Confidence bar visualization for top-3 predictions.
- Important symptom tags + model performance table.

## Step-by-Step Execution

### Step 1: Install dependencies
```bash
pip install -r backend/requirements.txt
```

### Step 2: Generate prepared dataset
```bash
python data/preprocess.py
```

### Step 3: Train and select best model
```bash
python training/train.py
```

### Step 4: Optional evaluation report
```bash
python training/evaluate.py
```

### Step 5: Start backend API
```bash
cd backend
uvicorn app:app --host 0.0.0.0 --port 8000
```

### Step 6: Start frontend
```bash
cd frontend
python -m http.server 5500
```

### Step 7: Open app
- Frontend: `http://localhost:5500`
- API docs: `http://localhost:8000/docs`

## API Example

### Request (`POST /predict`)
```json
{
  "age": 28,
  "gender": "female",
  "symptoms": ["fever", "cough", "headache"],
  "duration_days": 2,
  "lifestyle": {
    "smoking": false,
    "alcohol": false
  }
}
```

### Response
```json
{
  "predictions": [
    {"disease": "Influenza", "confidence": 0.81},
    {"disease": "Common Cold", "confidence": 0.62},
    {"disease": "COVID-19", "confidence": 0.39}
  ],
  "important_symptoms": ["fever", "cough", "fatigue"],
  "input_summary": {
    "age": 28,
    "gender": "female",
    "duration_days": 2,
    "symptom_count": 3
  }
}
```

## Troubleshooting
- **`model.pkl not found`** → run `python training/train.py` first.
- **Frontend shows API errors** → ensure backend is running at `http://localhost:8000`.
- **Dependency install fails** → verify network/proxy and Python version.
