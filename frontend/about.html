About This Final Year Project

This system demonstrates how machine learning can assist early disease prediction from patient symptoms and basic health profile.

Project Objectives
  • Predict top-3 likely diseases from symptoms.
  • Provide confidence scores for each class.
  • Show explainability through important symptoms.
  • Deliver as complete web-based CDSS.
Technology Stack
  • Frontend: HTML, CSS, JavaScript, Bootstrap
  • Backend: FastAPI + Pydantic
  • ML: scikit-learn, XGBoost
  • Data: Synthetic healthcare dataset pipeline
