About the Final Year Project

Production-style implementation of a complete clinical decision support pipeline.

Core Modules
  • Data generation + cleaning + encoded feature selection
  • Model training and comparison (LR, RF, SVM, XGBoost)
  • Best-model persistence and evaluation reporting
  • FastAPI backend with robust validation and error responses
  • Frontend dashboard with 5 independent functional pages
How to Use
  1. Generate data from data/preprocess.py
  2. Train from training/train.py
  3. Start API server and open frontend
  4. Predict diseases and inspect explainability outputs
